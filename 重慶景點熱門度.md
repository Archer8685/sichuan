# 重慶景點熱門度排名（砍天數參考用）

> 說明：景點類在資料庫沒有評分數字，以下熱門度是依旅遊攻略（小紅書／抖音／攜程／馬蜂窩）曝光度與遊客共識分級。行程目前保留 4 天（12/5–12/8）未變動，此清單供你判斷若砍成 3 天要留哪些。

## 🔥 S 級｜重慶天花板地標（砍了會後悔，務必留）

| 景點 | 目前排 | 熱門原因 |
|---|---|---|
| **洪崖洞** | D3 傍晚 | 重慶第一地標，吊腳樓夜景神似《千與千尋》，社群曝光度全城最高；免費 |
| **長江索道** | D4 | 「空中看兩江」的獨家體驗，過江時俯瞰渝中半島，網紅必搭 |
| **李子壩輕軌穿樓** | D2 | 抖音爆紅代表畫面——輕軌從居民樓中間穿過，重慶魔幻城市的招牌 |
| **磁器口古鎮** | D2 | 重慶第一古鎮，陳麻花、酸辣粉、老街小吃，知名度極高 |

## ⭐ A 級｜高人氣（建議留）

| 景點 | 目前排 | 熱門原因 |
|---|---|---|
| **南山一棵樹觀景台** | D4 | 拍重慶夜景最經典的機位，整個渝中半島燈火盡收 |
| **鵝嶺二廠文創公園** | D2 | 《從你的全世界路過》取景地，工業風文青打卡點 |
| **解放碑商圈** | D1 | 市中心地標，八一路好吃街，抵達日順路 |
| **千廝門大橋** | D3 | 拍洪崖洞全景的最佳機位（其實是洪崖洞的配套點）|
| **觀音橋＋不夜九街** | D2 | 江北夜生活地標，通宵夜市、酒吧、深夜食堂 |

|---|---|---|

## 💡 若砍成 3 天的建議

**核心原則：4 個 S 級全保住，砍掉 C 級與部分 B 級。**

保留：洪崖洞、長江索道、李子壩穿樓、磁器口（S）＋南山夜景、鵝嶺二廠、解放碑、觀音橋/不夜九街（A）

可砍：湖廣會館、羅漢寺、朝天門、來福士觀景台、下浩里、人民大禮堂、十八梯、三峽博物館（B/C）

**建議 3 天結構：**
- **Day 1（12/5 抵達半天）**：解放碑 → 洪崖洞黃昏夜景（千廝門大橋機位）
- **Day 2（12/6）**：李子壩穿樓 → 鵝嶺二廠 → 磁器口 → 觀音橋 + 不夜九街
- **Day 3（12/7）**：長江索道過江 → 南山一棵樹夜景（渝中—南岸經典動線）

⚠️ 注意：12/7 是週一，三峽博物館閉館——3 天版剛好去不成，但它是 B 級，不可惜。
